# TryHackMe: Pre-Security Path - Notes & Study Material

Welcome to my **TryHackMe Pre-Security Learning Path** repository. This repository serves as a comprehensive collection of structured notes, visual infographics, and text cheat sheets compiled while completing the Pre-Security path on TryHackMe.

---

## 📌 Repository Contents

The repository is organized into three main modules:

### 1. 📝 Handwritten Notes (`/Handwritten Notes`)
Detailed markdown files covering core concepts, room-by-room walkthroughs, and foundational learning modules:
* Cyber Security Roadmaps & Career Paths (Offensive vs. Defensive)
* Computer Hardware & Hardware Process Basics
* Server-Client Architecture & Network Basics
* Virtualization & Cloud Computing Fundamentals
* Operating System Introductions (Windows & Linux CLI Basics)
* OS Security Fundamentals

### 2. 📄 Text Cheat Sheets (`/Cheat Sheet .txt`)
Quick reference text files containing commands, syntax, shortcuts, and operational steps:
* Linux & Windows CLI Commands & Shortcuts
* Linux File Structure & Permissions
* Windows File System & Windows Sysinternals Overview
* Fundamental Bash Scripting Concepts

### 3. 🖼️ Visual Cheat Sheets & Infographics (`/Cheat sheet .png`)
High-resolution images and visual notes summarizing key concepts for quick revision:
* Cyber Security Career Paths
* Server-Client & Network Architecture
* OS Security, Linux Commands, and Windows Internals

---

## 🎯 Key Topics Covered

* **Cyber Security Fundamentals:** Basic introduction to Offensive/Defensive Security and Career Paths.
* **Computer Systems & Architecture:** Client-Server models, Hardware processes, Virtualization, and Cloud Computing.
* **Linux Fundamentals:** Terminal operations, File system navigation, Basic CLI commands, and Bash scripting.
* **Windows Fundamentals:** CMD utilities, File system structures, Shortcuts, and Sysinternals tools.
